# Windows PowerShell для Office Web Apps (SharePoint 2013)

 

_**Применимо к:**Office Web Apps, SharePoint Foundation 2013, SharePoint Server 2013_

_**Последнее изменение раздела:**2016-12-16_

**Сводка.** Поиск статей о командлетах Windows PowerShell SPWOPI, которые служат для настройки сервера Office Web Apps.

**Аудитория:** ИТ-специалисты

При локальном использовании SharePoint 2013Office Web Apps предлагает обновленные версии приложений Word Web App, Excel Web App, PowerPoint Web App и OneNote Web App. Благодаря этому пользователи могут просматривать и, при наличии соответствующих прав, редактировать документы Office с использованием поддерживаемого веб-браузера на компьютерах и мобильных устройствах, таких как телефоны под управлением Windows Phone, iPhone и iPad.

![Логотип Office 2013](images/JJ219456.a106e261-2cd0-43b7-af77-92de7e4b6fb9(Office.15).png "Логотип Office 2013")В следующей таблице перечислены и описаны статьи для каждого командлета Windows PowerShell SPWOPI, который настраивает Office Web Apps для использования вместе с SharePoint 2013.

<td><p>Возвращает список привязок, созданных с помощью <strong>New-SPWOPIBinding</strong> в текущей ферме SharePoint, где выполняется данный командлет.</p></td>
</tr>
<tr class="even">
<td><p><a href="get-spwopisuppressionsetting.md">Get-SPWOPISuppressionSetting</a></p></td>
<td><p>Возвращает параметры подавления для текущей фермы SharePoint, где выполняется данный командлет.</p></td>
</tr>
<tr class="odd">
<td><p><a href="get-spwopizone.md">Get-SPWOPIZone</a></p></td>
<td><p>Возвращает зону, настроенную в текущей ферме SharePoint, которая будет использоваться приложением WOPI.</p></td>
</tr>
<tr class="even">
<td><p><a href="new-spwopibinding.md">New-SPWOPIBinding</a></p></td>
<td><p>Создает новую привязку для связи расширений имен файлов или приложений с действиями в текущей ферме SharePoint, где выполняется этот командлет.</p></td>
</tr>
<tr class="odd">
<td><p><a href="new-spwopisuppressionsetting.md">New-SPWOPISuppressionSetting</a></p></td>
<td><p>Отключает Office Web Apps для действия и типа документа или привязки, указанной в текущей ферме SharePoint.</p></td>
</tr>
<tr class="even">
<td><p><a href="remove-spwopibinding.md">Remove-SPWOPIBinding</a></p></td>
<td><p>Удаляет привязки приложений, расширений имен файлов и связанных с ними действий в текущей ферме SharePoint, где выполняется этот командлет.</p></td>
</tr>
<tr class="odd">
<td><p><a href="remove-spwopisuppressionsetting.md">Remove-SPWOPISuppressionSetting</a></p></td>
<td><p>Удаляет настройки подавления для типа файла или программного идентификатора (ProgID) в текущей ферме SharePoint, в которой выполняется этот командлет.</p></td>
</tr>
<tr class="even">
<td><p><a href="set-spwopibinding.md">Set-SPWOPIBinding</a></p></td>
<td><p>Обновляет действие по умолчанию при щелчке мышью для приложения или привязки расширения имени файла.</p></td>
</tr>
<tr class="odd">
<td><p><a href="set-spwopizone.md">Set-SPWOPIZone</a></p></td>
<td><p>Выполняет настройку зоны, используемой текущей фермой SharePoint для перехода в браузере к приложению WOPI.</p></td>
</tr>
<tr class="even">
<td><p><a href="update-spwopiproofkey.md">Update-SPWOPIProofKey</a></p></td>
<td><p>Обновляет открытый ключ, используемый для подключения к приложению WOPI в текущей ферме SharePoint, где выполняется данный командлет.</p></td>
</tr>
</tbody>
</table>
